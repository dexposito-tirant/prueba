Presentación

Bienvenidos al módulo de Lenguaje de Marcas. En este curso, exploraremos los fundamentos y aplicaciones de los lenguajes de marcado, centrándonos en HTML y CSS. Estos lenguajes son la base del desarrollo web y son esenciales para crear sitios web modernos y atractivos.

Objetivos del Curso

  • Comprender la estructura y el propósito de HTML.
  • Aprender a utilizar las etiquetas HTML para crear contenido estructurado.
  • Introducir los conceptos básicos de CSS para el diseño y la presentación.
  • Desarrollar habilidades prácticas a través de ejercicios y proyectos.

Materiales y Recursos

Para aprovechar al máximo este curso, recomendamos tener acceso a las siguientes herramientas y recursos:

  • Un editor de código, como Visual Studio Code o Sublime Text.
  • Diferentes navegadores web (Chrome, Firefox, etc.) para probar el código.
  • Recursos en línea, como MDN Web Docs y W3Schools.

Estructura del Curso

El curso está dividido en varias secciones, cada una enfocada en diferentes aspectos del desarrollo web:

  • Introducción: Conceptos básicos y la importancia del lenguaje de marcas.
  • HTML: Historia, etiquetas básicas, y estructura de documentos HTML.
  • CSS: Selección de elementos, propiedades de estilo, y diseño responsivo.
  • JavaScript: Fundamentos de la programación y cómo añadir interactividad a las páginas web.

Evaluación

La evaluación del curso se basará en la participación en clase, ejercicios prácticos, y un proyecto final donde demostrarás tus habilidades adquiridas.

Dentro de cada unidad formativa, se realizarán difentes ejercicios prácticos de seguimiento de la unidad. Dichos ejercicios no son evaluables, pero serán corregidos en el aula y es muy recomendable realizarlos para poder superar dicha unidad.

La nota de cada unidad formativa se obtendrá de una prueba objetiva individual (examen), que será el 60% de la nota de la unidad formativa y el 40% sera del proyecto propuesto para dicha unidad.

Importante: La nota mínima de los exámenes para poder hacer media es de 5. En cualquier otro caso, se deberá recuperar la unidad formativa a final de curso.

Para obtener la nota del proyecto, es imprescindible superar una prueba de validación individual. Esta prueba tiene como objetivo asegurar que cada estudiante ha comprendido y dominado los conceptos clave desarrollados durante el proyecto.

Importante: Si no se supera esta prueba de validación, la calificación del proyecto será un 0, independientemente del trabajo realizado en grupo o de la calidad del proyecto entregado.

Por lo tanto, es crucial que cada miembro del equipo domine los aspectos fundamentales y sea capaz de defender su trabajo individualmente.

Para más información, se debe consultar la programación didáctica de la asignatura.
