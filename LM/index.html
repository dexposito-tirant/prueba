Introducción

Bienvenidos al módulo de Lenguaje de Marcas. En este curso, exploraremos los fundamentos y aplicaciones de los lenguajes de marcado, centrándonos en HTML y CSS. Estos lenguajes son la base del desarrollo web y son esenciales para crear sitios web modernos y atractivos.

Objetivos del Curso

  • Comprender la estructura y el propósito de HTML.
  • Aprender a utilizar las etiquetas HTML para crear contenido estructurado.
  • Introducir los conceptos básicos de CSS para el diseño y la presentación.
  • Desarrollar habilidades prácticas a través de ejercicios y proyectos.

Materiales y Recursos

Para aprovechar al máximo este curso, recomendamos tener acceso a las siguientes herramientas y recursos:

  • Un editor de código, como Visual Studio Code o Sublime Text.
  • Navegadores web modernos (Chrome, Firefox, etc.) para probar el código.
  • Recursos en línea, como MDN Web Docs y W3Schools.

Estructura del Curso

El curso está dividido en varias secciones, cada una enfocada en diferentes aspectos del desarrollo web:

  • Introducción: Conceptos básicos y la importancia del lenguaje de marcas.
  • HTML: Historia, etiquetas básicas, y estructura de documentos HTML.
  • CSS: Selección de elementos, propiedades de estilo, y diseño responsivo.
  • JavaScript: Fundamentos de la programación y cómo añadir interactividad a las páginas web.
  • Bootstrap: Introducción al framework CSS más popular para desarrollo responsive.
  • Sass: Preprocesador CSS que facilita la escritura de estilos más eficientes y organizados.

Evaluación

La evaluación del curso se basará en la participación en clase, ejercicios prácticos, y un proyecto final donde demostrarás tus habilidades adquiridas.

¿Qué es un Lenguaje de Marcas?

Un lenguaje de marcas es un sistema que utiliza etiquetas para anotar un documento de una manera que es sintácticamente distinguible del texto. Estas anotaciones típicamente sirven para indicar cómo se debe presentar el documento en el navegador, estructurarlo, o incluso describir los datos que contiene.

Ejemplos de Lenguajes de Marcas

  • HTML (HyperText Markup Language): Utilizado para crear y estructurar páginas web.
  • XML (eXtensible Markup Language): Usado para transportar y almacenar datos.
  • Markdown: Un lenguaje ligero de marcas para formatear texto.

Características de los Lenguajes de Marcas

  • Uso de etiquetas para definir elementos.
  • Estructura jerárquica de los documentos.
  • Facilidad de lectura y edición tanto por humanos como por máquinas.
  • Independencia de plataforma y sistema operativo.

Importancia de los Lenguajes de Marcas

Los lenguajes de marcas son fundamentales en el desarrollo web y en la gestión de datos. Permiten a los desarrolladores definir la estructura y presentación de documentos, lo que es esencial para la creación de sitios web accesibles y bien organizados. Además, facilitan la interoperabilidad entre diferentes sistemas y plataformas.

HTML

Historia de HTML

HTML, o HyperText Markup Language, es el estándar de marcado utilizado para crear páginas web. Fue creado por Tim Berners-Lee en 1991 y ha evolucionado a través de varias versiones, la más reciente siendo HTML5.

Etiquetas Básicas

Las etiquetas básicas de HTML incluyen <html>, <head>, <body>, <title>, <h1> a <h6>, <p>, <a>, <div>, y <span>, entre otras. Estas etiquetas permiten estructurar el contenido y definir la jerarquía del documento.

CSS

Conceptos Básicos de CSS

CSS, o Cascading Style Sheets, es el lenguaje utilizado para describir la presentación de documentos HTML. Permite aplicar estilos como colores, fuentes, márgenes y layout a los elementos HTML.

Selección de Elementos

CSS utiliza selectores para aplicar estilos a elementos específicos. Algunos ejemplos incluyen selectores de tipo, selectores de clase, selectores de ID y selectores de atributo.

JavaScript

Introducción a JavaScript

JavaScript es un lenguaje de programación que permite añadir interactividad a las páginas web. Puede utilizarse para crear dinámicamente contenido, validar formularios, manejar eventos y mucho más.

Manipulación del DOM

El Document Object Model (DOM) es una representación estructurada del documento HTML. JavaScript puede interactuar con el DOM para cambiar el contenido, estructura y estilos de la página de manera dinámica.

Bootstrap

¿Qué es Bootstrap?

Bootstrap es un framework de CSS de código abierto dirigido al desarrollo de sitios y aplicaciones web responsivas. Contiene plantillas de diseño basadas en HTML y CSS para tipografía, formularios, botones, navegación y otros componentes de interfaz.

Componentes Principales de Bootstrap

Bootstrap ofrece una amplia gama de componentes reutilizables como botones, tarjetas, modales, menús desplegables y mucho más, todos diseñados para ser responsivos y personalizables.

Sass

¿Qué es Sass?

Sass (Syntactically Awesome Stylesheets) es un preprocesador de CSS que añade características avanzadas como variables, anidamiento, herencia y mixins, lo que facilita la escritura y el mantenimiento de estilos CSS.

Características de Sass

Entre las características destacadas de Sass se incluyen la posibilidad de usar variables para almacenar valores, anidar reglas CSS de manera jerárquica, y crear mixins para reutilizar bloques de código CSS.
